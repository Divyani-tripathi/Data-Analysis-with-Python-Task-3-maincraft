# 🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project is part of my **Data Analysis with Python Internship** at **MainCrafts Technology**.

The objective of this project is to perform **Exploratory Data Analysis (EDA)** on the Titanic dataset to understand the factors that influenced passenger survival. The project includes data cleaning, feature engineering, statistical analysis, and data visualization using Python.

---

## 🎯 Objectives

- Load and explore the Titanic dataset
- Handle missing values
- Perform feature engineering
- Analyze survival patterns
- Visualize important relationships between variables
- Extract meaningful insights from the data

## 🔍 Key Insights

- Approximately **38%** of passengers survived.
- Female passengers had a significantly higher survival rate than male passengers.
- First-class passengers had the highest survival rate.
- Children showed better survival chances than adults and senior passengers.
- Passengers travelling in small families generally had higher survival rates.
- Multiple factors collectively influenced passenger survival.
